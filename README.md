# Simulador de Otimização de Carregadores Elétricos

Este projeto simula diferentes estratégias de uso de carregadores para veículos elétricos, avaliando impacto em:

- Tempo médio de espera
- Ociosidade das vagas

## Versões do simulador

- V1: Sem sistema
- V2: Apenas software (fila inteligente)
- V3: Apenas hardware (detecção automática)
- V4: Sistema híbrido (software + hardware)
- V5: Sistema completo (inclui expansão de carregadores)

## Objetivo

Demonstrar como a combinação de tecnologia e gestão pode aumentar significativamente a eficiência da infraestrutura de recarga.

## Metodologia

Os resultados apresentados são médias de 5 execuções independentes da simulação.
